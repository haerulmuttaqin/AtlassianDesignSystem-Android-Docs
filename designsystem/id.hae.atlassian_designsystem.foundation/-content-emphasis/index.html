ContentEmphasis

enum ContentEmphasis : Enum<ContentEmphasis>

Content emphasis changes content color in a semantic way.

The main use case is to change ContentColors.normal (InkNormal) color other content/ink shades, though the logic is able to handle also other colors - in such case the color resolution applies opacity.
